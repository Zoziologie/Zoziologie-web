eBirdTarget2csv

eBirdTarget2csv is a very simple JavaScript app to convert the eBird target webpage (html) to a csv file.

How does it work?

  1. Go on the eBird target page.
  2. Generate the target list you want on the webpage.
  3. Save the html page in your computer (Ctrl+S on windows or ⌘+S on mac).
  4. Upload the html file in box below (drag and drop work fine).
    5. Drop the html file here or click to upload
  5. This will open a "save as" windows to save your csv file generated from the html.
