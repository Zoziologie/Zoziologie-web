Improved Biolovision Visualisation

What is iBV?

iBV stands for "Improved Biolovision Visualisation". It's a web-app designed to visualize the data from all biolovision websites.

What can I do with iBV?

  • Query the database on a specific area by drawing a rectangle on the amp
  • View the query on a zoomable map (only available with certain websites: ornitho.ch, ornitho.cat...)
  • Create a spreadsheet with the species list from the query

Where to start?

Check out the instructions to learn how to use the website. If you're a pro, start directly on the search menu

How to use the search menu?

  1. Place:
    • The first row defines the database query. You can change it by searching by name (left) or select it on the map with the button "Select"
    • The second row allows you to either select the whole database "All" or draw on the map a rectangle in which the query will be made "Coordinate". Click on the "draw" button to make a new rectangle.
  2. Date: self-explanatory... the second left option is for all years but limited in the season.
  3. Species: Choose either a set of species defined by rarity: +++ (very rare) to --- (very common) and E for exotic or a specific one (bottom search). The "Info" will bring you to the current selected database website at the page showing you the distribution of the selected species.(very useful IMO). You can search in French or English. Other language are available if you need them.
  4. Other: Limit your search to these option.
  5. Output: Choose the output of your :
    • Various visualization of the selected databse. This is basically generating a link to the website.
    • "View On Map" to view results on a current zoomable map. Not all database allows an export in kml (You can search them in the Place menu and "Select" button)
    • "Export to CSV" to generate the table of the specie list
